# Prompt: Create Munro Template

Using docs/plan.md, create a clean and reusable Markdown template for a Munro entry. it should be kept under munros directory

It should include:
- Completion status
- Date
- Completion number (chronological order, e.g. 1 for first completed)
- Weather
- Rating
- Companions (comma separated list of individuals and case insensitive)
- Notes
- A small "moment" section
- Placeholders for summit and route photos

Keep it simple, consistent, and easy to maintain.
